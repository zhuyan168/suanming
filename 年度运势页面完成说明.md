## 🚀 如何访问

1. **从主页访问**:
   - 启动开发服务器: `npm run dev`
   - 打开浏览器访问: `http://localhost:3000`
   - 点击"运势测算"模块中的"年度运势"按钮

2. **直接访问**:
   - 年度运势入口: `http://localhost:3000/fortune/annual`
   - 塔罗年度: `http://localhost:3000/fortune/annual/tarot`
   - 生肖易经: `http://localhost:3000/fortune/annual/eastern`
