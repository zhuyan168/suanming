## 🚀 如何使用

### 开发环境运行

```bash
# 安装依赖（如果还没安装）
npm install

# 启动开发服务器
npm run dev
```

访问：`http://localhost:3000`

### 测试流程

1. **打开主页** → 看到"是否占卜"模块
2. **点击"掷筊占卜 (Moon Blocks)"按钮**
3. **进入掷筊页面** → 查看介绍和两块筊杯
4. **点击"开始掷筊"** → 观看掷筊动画（2秒）
5. **自动跳转到结果页** → 查看占卜结果
6. **选择操作**：
   - 点击"再掷一次" → 返回掷筊页面
   - 点击"返回首页" → 回到网站首页
