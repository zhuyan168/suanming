# 🎴 掷筊占卜 - 使用说明

## 🚀 快速开始

### 1. 启动项目

```bash
npm run dev
```

访问：`http://localhost:3000`

### 2. 进入掷筊页面

**方式一：** 从主页点击「掷筊占卜 (Moon Blocks)」按钮

**方式二：** 直接访问 `/divination/jiaobei`

---

## 🎯 使用流程

### Step 1: 默念问题
在心中默念你想要询问的问题（建议是可以用"是/否"回答的问题）

**示例问题：**
- 我应该接受这份工作吗？
- 这段感情适合继续发展吗？
- 今天适合做重要决定吗？
- 我的计划会顺利进行吗？

### Step 2: 点击「开始掷筊」

点击页面底部的紫色按钮

### Step 3: 观看动画

动画时长 **1.8 秒**，展示完整的掷筊过程：

1. **抛起**（0-0.27s）
   - 两块圣杯从底部弹起

2. **上升 + 旋转**（0.27-0.72s）
   - 圣杯旋转上升至最高点
   - 三轴旋转（前后、左右、平面）

3. **下落**（0.72-1.53s）
   - 重力下落
   - 继续旋转

4. **落地 + 弹跳**（1.53-1.8s）
   - 落地瞬间光晕扩散
   - 轻微弹跳
   - 显示最终朝向

### Step 4: 查看结果

动画结束后，页面会显示结果：

#### 🌕🌑 圣筊（一正一反）
**含义：** 此事可行，神明允准
**建议：** 顺势而为，把握时机

#### 🌕🌕 阴筊（两正）
**含义：** 暂缓行事，不宜进行
**建议：** 再思考，等待更好时机

#### 🌑🌑 笑筊（两反）
**含义：** 神明含笑未答
**建议：** 重新整理问题，再问一次

### Step 5: 自动跳转

结果显示 **1.5 秒** 后，自动跳转到详细结果页

## 💡 使用技巧

### 1. 问题设定

**✅ 推荐的问题类型：**
- 具体明确的问题
- 可以用是/否回答
- 近期关心的事项
- 需要决策的事情

**❌ 不推荐的问题：**
- 过于宽泛（"我的未来如何？"）
- 多重选项（"A 好还是 B 好？"）
- 已知答案的问题
- 测试性质的问题

### 2. 解读结果

#### 圣筊 🌕🌑
- **信号：** 绿灯，可以前进
- **态度：** 积极乐观，把握机会
- **行动：** 顺势而为，勇敢尝试

#### 阴筊 🌕🌕
- **信号：** 红灯，暂时停止
- **态度：** 谨慎思考，不急于行动
- **行动：** 暂缓计划，寻找更好时机

#### 笑筊 🌑🌑
- **信号：** 黄灯，需要重新评估
- **态度：** 问题可能不够明确
- **行动：** 重新整理思路，再次询问

### 3. 多次掷筊

如果获得「笑筊」，建议：
1. 重新整理问题
2. 确保问题清晰具体
3. 调整心态后再问
4. 最多问 3 次

### 4. 结合理性思考

占卜结果**仅供参考**，应该：
- ✅ 结合实际情况判断
- ✅ 作为决策参考之一
- ✅ 保持理性思考
- ❌ 不要完全依赖占卜

## 🎭 文化背景

### 什么是掷筊？

掷筊（jiǎo bēi），又称「问筊」或「掷杯」，是中华传统民俗中与神明沟通的方式。

### 历史渊源

- 起源于道教和民间信仰
- 用于重要决策前征询神意
- 广泛流行于中国南方及台湾地区

### 传统用法

1. 到庙宇上香祈福
2. 心中默念问题
3. 掷出两块筊杯
4. 根据结果判断神明回应
5. 重要事项会连问三次确认

### 现代应用

- 现代科技让传统文化得以传承
- 随时随地体验传统占卜
- 保持仪式感和神秘感
- 作为心理辅助工具

## 🔧 技术支持

### 常见问题

**Q: 动画卡顿怎么办？**
A: 建议使用现代浏览器（Chrome、Firefox、Safari、Edge）

**Q: 按钮点击无反应？**
A: 检查是否在动画进行中，等待动画结束后再点击

**Q: 结果页无法跳转？**
A: 刷新页面重试，或检查浏览器控制台错误信息

**Q: 移动端显示异常？**
A: 建议在竖屏模式下使用，确保网络连接正常
